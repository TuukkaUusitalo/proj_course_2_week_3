## To setup:
- clone repository:
  - git clone https://github.com/TuukkaUusitalo/proj_course_2_week_3.git /polku/tyhjaan/kansioon
  - go to the project root to run it
  - You have to have MariaDB and MySQL, otherwise the database cannot be created
  - To run the javafx UI, you can still run it with command mvn javafx:run
